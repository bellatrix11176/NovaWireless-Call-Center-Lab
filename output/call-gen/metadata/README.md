# output/call-gen/metadata/

This folder contains the structured call metadata files — one CSV per month covering all 12 months of the simulation.

## Files

| File | Description |
|------|-------------|
| `calls_metadata_2025-01.csv` through `calls_metadata_2025-12.csv` | Structured call records for each month including scenario type, agent ID, customer ID, resolution flags, credit details, repeat contact flags, and churn risk |

Each file contains approximately 8,000 calls and is ~4MB.

## These files are not in the repo

They are generated by the pipeline and excluded from git. Download them from the [v1.0 Release](https://github.com/bellatrix11176/NovaWireless-Call-Center-Lab/releases/tag/v1.0) and place them in this folder.

Or regenerate them by running:

```bash
python run_all.py
```
