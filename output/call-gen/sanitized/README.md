# output/call-gen/sanitized/

This folder contains the sanitized and validated call records — one CSV per month ready for downstream analysis.

## Files

| File | Description |
|------|-------------|
| `calls_sanitized_2025-01.csv` through `calls_sanitized_2025-12.csv` | Validated call records with enriched outcome flags, governance signals, trust scores, and rep-level aggregates |

Each file contains approximately 8,000–11,000 calls and is ~25MB.

## How sanitized differs from metadata

The metadata files contain raw simulation output. The sanitized files have been processed by `02_sanitize_calls.py` which validates credit types, computes outcome flags, joins transcript data, and enriches each record with governance signal inputs including DAR, DOV, and trust decay.

## These files are not in the repo

They are generated by the pipeline and excluded from git. Download them from the [v1.0 Release](https://github.com/bellatrix11176/NovaWireless-Call-Center-Lab/releases/tag/v1.0) and place them in this folder.

Or regenerate them by running:

```bash
python run_all.py
```
