# data/output/

This folder contains output files from companion pipelines in the NovaWireless ecosystem.

## Files

| File | Size | Description |
|------|------|-------------|
| `calls.csv` | ~18MB | Call records in CSV format |
| `calls.json` | ~79MB | Call records in JSON format |
| `signals.csv` | ~44KB | Governance signal outputs in CSV format |
| `signals.json` | ~168KB | Governance signal outputs in JSON format |

## These files are not in the repo

They are excluded from git. Download them from the 
[v1.0 Release](https://github.com/bellatrix11176/NovaWireless-Call-Center-Lab/releases/tag/v1.0)
and place them in this folder.
