# data/

This folder contains the core synthetic datasets used across the NovaWireless Call Center Lab pipeline.

## Files

| File | Size | Description |
|------|------|-------------|
| `customers.csv` | ~14MB | Synthetic customer population (100,000 customers) with demographics, account details, and churn risk scores |
| `master_account_ledger.csv` | ~8MB | Master account ledger used by the call generation pipeline |
| `novawireless_employee_database.csv` | ~1MB | Call center rep workforce with individual performance profiles |

## Subfolders

| Folder | Description |
|--------|-------------|
| `employee_generation_inputs/` | Small reference CSVs used to generate rep personas — included in the repo |
| `external/params_sources/` | Large external parameter files — download from release |
| `output/` | Output files from companion pipelines |

## These files are not in the repo

They are generated by the pipeline and excluded from git. Download them from the [v1.0 Release](https://github.com/bellatrix11176/NovaWireless-Call-Center-Lab/releases/tag/v1.0) and place them in this folder.

Or regenerate them by running:

```bash
python run_all.py
```
